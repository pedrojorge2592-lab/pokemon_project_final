# Pokémon Project – MLOps & Project Management 🐉

A future-proof MLOps mini-project that predicts whether a Pokémon is **Legendary** based on its stats and types.

---

## 🧰 Tech Stack
- Python 3.13
- scikit-learn for ML
- MLflow for experiment tracking
- Streamlit (UI)
- FastAPI (API)
- Docker for packaging
